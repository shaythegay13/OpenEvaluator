================================================================================
WHY THESE IMPROVEMENTS
================================================================================

1. DESKEWING
   - George's sketches are often rotated 30-45°
   - Vision API struggles with rotated text
   - Auto-correction makes OCR 40%+ more accurate

2. UPSCALING (2x)
   - Small handwritten measurements are hard to read
   - Upscaling before OCR makes features 4x larger
   - Helps Vision API detect fine details

3. ADAPTIVE THRESHOLDING
   - Hand-drawn sketches have varying brightness
   - Fixed threshold loses information in shadows
   - Adaptive approach preserves detail

4. DOCUMENT_TEXT_DETECTION
   - Designed for documents with mixed text/drawings
   - Returns structured data (coordinates, confidence)
   - Better for extracting measurements from sketches
