## Notes

- **Merge vs. Individual**: The merged PDF is for the evaluator's records. The individual pages must be used for official submittal, as wet signatures and LPI stamps are applied per page.
- **Correction Form Link**: Replace `[CORRECTION FORM LINK]` with the actual URL to the Google Form or equivalent correction request form.
- **Conditional Attachments**: Only include HHE-204, GSB Soil Notes, and/or HHE-300A when triggered by the application data.
- **System Type Detection**: The body should dynamically reflect whether Pages 3+4 or Pages 7+8 were used based on the system type and component selections.
